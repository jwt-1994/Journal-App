## 背景模版库

### 功能概述
提供手账背景页模版库，包含内置配色/纹理和用户自定义上传。

### 内置模版
- 系统首次启动时自动初始化内置模版
- 纯色背景：20+ 预设配色（白、米白、浅粉、浅蓝、牛油果绿、奶油黄、薰衣草紫、浅灰、卡其等）
- 材质纹理：10+ 纹理（牛皮纸、网格、水彩、布纹、木纹、大理石纹、点阵、方格、横线、空白）
- 内置模版不可删除

### 用户背景
- 支持上传 JPG/PNG/WEBP 图片作为背景
- 自动缩放至 1920x1080 标准尺寸
- 自动生成缩略图（320x180）
- 用户背景可删除
- 文件大小限制 5MB

### API
- `GET /api/backgrounds?type=preset|user` — 列表查询
- `POST /api/backgrounds` — 上传背景图片
- `DELETE /api/backgrounds/:id` — 删除背景（仅限用户自定义）
- `GET /api/backgrounds/:id/file` — 获取背景图片
- `POST /api/backgrounds/presets/init` — 初始化内置模版

### 前端
- 背景库页面：网格展示所有背景，分类筛选（纯色/纹理/我的）
- 上传按钮：选择图片上传为背景
- 背景预览：点击查看大图
- 删除按钮：仅自定义背景可删除

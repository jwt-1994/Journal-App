## 拼贴画布

### 功能概述
基于 Canvas 的拖拽布局画布，支持素材自由组合排版，保存拼贴方案并导出为图片。

### 画布功能
- 选择背景后进入拼贴编辑模式
- 左侧素材库面板：显示所有素材的缩略图列表，支持搜索和分类筛选
- 中央画布区域：react-konva 实现的无限画布，支持缩放和平移
- 右侧图层面板：显示当前画布上所有素材的图层列表，支持拖拽调整顺序
- 拖拽素材从面板到画布放置
- 选中素材后显示变换控件（移动、缩放、旋转）
- 双击素材可删除
- 支持撤销/重做操作（基于布局历史快照）

### 方案管理
- 保存方案：命名 + 保存当前画布状态
- 方案列表：查看所有已保存方案
- 加载方案：打开方案进入编辑模式
- 更新方案：编辑后保存更新
- 删除方案：删除方案及其预览图
- 方案数据格式 JSON：
```json
[
  {
    "material_id": 1,
    "x": 100,
    "y": 200,
    "width": 150,
    "height": 150,
    "rotation": 0,
    "zIndex": 0
  }
]
```

### 导出功能
- 导出为 PNG（默认）或 JPEG
- 导出尺寸为画布原始尺寸
- 支持自定义缩放比例（0.5x, 1x, 2x）
- 使用 Canvas toDataURL 导出

### API
- `GET /api/collages` — 方案列表
- `POST /api/collages` — 创建方案
- `GET /api/collages/:id` — 获取方案详情
- `PUT /api/collages/:id` — 更新方案
- `DELETE /api/collages/:id` — 删除方案

### 前端
- 拼贴页面：路由 `/collage`
- 拼贴编辑器：三栏布局（素材面板 | 画布 | 图层面板）
- 顶部工具栏：保存、加载、导出、撤销、重做
- 方案列表弹窗：选择已保存方案加载或删除

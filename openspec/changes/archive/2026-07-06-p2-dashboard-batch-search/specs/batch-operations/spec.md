# batch-operations

## 概述
素材库支持多选素材后的批量抠图和批量删除操作。

## 需求

### 1. 批量操作模式切换
- 素材库顶部工具栏增加"批量操作"按钮
- 点击后进入批量操作模式，每张卡片/列表项出现复选框
- 右键区域显示"全选当前页"和"取消"按钮

### 2. 多选交互
- 点击卡片图片区域或复选框可选中/取消选中
- 支持全选/取消全选当前页
- 切换页面时清空选中状态
- 批量模式下隐藏单个卡片的操作按钮（查看/抠图/删除）

### 3. 批量抠图
- 选中素材后，底部浮动操作栏出现"批量抠图"按钮
- 点击后弹出确认对话框
- 确认后调用 `POST /api/materials/batch-remove-bg` 接口
- 后端跳过已抠图和处理中的素材，只触发未抠图和失败的
- 完成后显示触发数量

### 4. 批量删除
- 选中素材后，底部浮动操作栏出现"批量删除"按钮
- 点击后弹出二次确认对话框（红色危险按钮）
- 确认后调用 `POST /api/materials/batch-delete` 接口
- 后端同时删除文件系统中的原图和抠图文件

### 5. 浮动操作栏
- 固定在页面底部居中
- 显示已选数量
- 包含批量抠图和批量删除两个按钮
- 选中数量为 0 时自动隐藏

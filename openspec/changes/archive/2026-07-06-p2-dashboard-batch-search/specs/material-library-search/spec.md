# material-library (modified)

## 概述
在 P0 素材库基础上增加文件名搜索、抠图状态筛选、排序功能。

## 修改内容

### 1. 文件名搜索
- 素材库顶部增加搜索输入框（带搜索图标）
- 输入关键词后实时过滤，后端使用 SQL LIKE 模糊匹配 `original_name`
- 支持清空搜索

### 2. 抠图状态筛选
- 增加抠图状态下拉筛选器：全部/未抠图/处理中/已抠图/失败
- 与分类筛选可同时使用

### 3. 排序
- 增加排序下拉选择器
- 支持四种排序方式：上传时间最新/最旧、文件大小最大/最小
- 默认按上传时间最新排序

### 4. 后端接口扩展
- `GET /api/materials` 新增参数：`search`、`bg_status`、`sort_by`、`sort_order`
- 所有筛选/排序/搜索参数可自由组合

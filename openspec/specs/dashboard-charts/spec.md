# dashboard-charts

## 概述
报表看板使用 recharts 图表库替代 P0 的纯文本统计，提供分类占比饼图、上传趋势折线图和抠图状态柱状图。

## 需求

### 1. 概览统计卡片
- 显示素材总数、存储空间、已抠图数量、抠图失败数量四张卡片
- 已抠图卡片显示"已抠图数 / 总数"比率

### 2. 分类占比饼图
- 使用 PieChart 展示各分类素材数量占比
- 每个扇区显示分类名称和百分比
- 支持鼠标悬停查看详情

### 3. 抠图状态柱状图
- 使用 BarChart 展示四种抠图状态（未抠图/处理中/已抠图/失败）的数量分布
- 不同状态使用不同颜色区分

### 4. 上传趋势折线图
- 使用 LineChart 展示近 30 天每日上传数量
- X 轴为日期，Y 轴为上传数量
- 后端新增 `GET /api/dashboard/upload-trend` 接口

### 5. 最近上传表格
- 保留 P0 的最近上传列表，点击跳转素材库
